# Catalogo Curado de Referencias de Design

Use este catalogo na Fase 3 para selecionar as referencias mais relevantes para cada cliente.
NAO visite todos — filtre por tags que combinam com o conceito criativo do cliente.

## Como usar

1. Leia o conceito narrativo do cliente (Fase 2.2)
2. Identifique 2-3 tags dominantes (ex: "minimal + elegante + scroll-experience")
3. Filtre os sites abaixo que compartilham essas tags
4. Navegue nos 3-5 mais relevantes e anote CONCRETAMENTE:
   - Espacamento entre secoes (px)
   - Hierarquia tipografica (tamanhos, pesos, contrastes)
   - Tecnicas de animacao usadas (GSAP, CSS, Three.js, Lottie)
   - O que faz parecer premium (polish, detalhes, timing)
   - Layout patterns reutilizaveis para o cliente
